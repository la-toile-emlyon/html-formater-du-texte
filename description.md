Formater du texte
========================

## Objectif
Mettre en forme des informations, à partir d'un contenu qui vous est fourni, en utilisant les éléments HTML appropriés.

Etape par étape
-------------------

<h3>🔁 La routine</h3>
</summary>

1. Créer un nouveau dossier dans votre espace de travail
2. Créer un fichier `index.html`
3. Ajouter le squelette HTML de base (dans VS Code, `!` puis `Entrée`)

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  
</body>
</html>
```

### Consignes spécifiques
1. Ouvrir le [fichier `contenu.txt`](1.1-html-niveau-1/1.1.1-formater-du-texte/contenu.txt)
2. Insérer, au sein de l'élément `body`, le code html correspondant à chaque élément de contenu.
    - titres `h1`, `h2` etc.
    - paragraphes `p` et sauts de ligne `br`
    - liste ordonnée `ol` et non ordonnée `ul`, et éléments de liste `li`
    - contenu mis en avant en gras `strong` et en italique `em`
